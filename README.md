Meal Lab App

A desktop application built with JavaFX to search and organize recipes using TheMealDB API.

Features:
- Search recipes by Name, Ingredient, or ID.
- Save to Favorites & Cooked History (stored locally in JSON).
- Get a random meal suggestion instantly.

Tech Stack:
- Java 17 & JavaFX
- Maven (Dependency Management)
- Libraries: OkHttp, Jackson Databind, JUnit 5
